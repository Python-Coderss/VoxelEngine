PROJECT: Voxel Engine (Java 8 / LWJGL3 / OpenGL 4.3+)
OBJECTIVE: Incremental build of a ray-casted voxel engine using compute shaders.

--- ARCHITECTURE OVERVIEW ---
- Rendering: Pure compute shader ray casting (no rasterization/triangles).
- Output: Compute shader writes to an RGBA8 image2D, displayed via a fullscreen quad.
- Memory: All voxel and world data stored in SSBOs (Direct State Access utilized).
- World Structure: 16³ voxel chunks grouped into a 128³ Region Indirection Table (2048³ voxels total).
- Traversal: Two-level Hierarchical DDA (Macro jumps empty chunks, Micro marches non-empty ones).
- Camera: Custom orthonormal basis (Forward, Right, Up) with WASD + Mouse look + Space/Shift verticality.
- Biomes: Procedural temperature and humidity maps (Perlin noise) driving Minecraft-style colormap tinting.
- Materials: PBR-lite properties (transparency, reflectivity) and face-specific textures (6 textures per block).

--- DATA LAYOUTS (SSBO / Textures) ---
1. RegionIndirectionTable (Binding 1): [rx][ry][rz] -> uint slot | 0xFFFFFFFF (EMPTY)
2. ChunkPool (Binding 2): [slot] -> 16³ uint voxel cells.
3. EntityParts (Binding 3): flat array of EntityPart structs (worldToLocal matrix + offsets).
4. EntityVoxelPool (Binding 4): storage for rotating entity voxel data.
5. LightPool (Binding 5): Packed RGB lighting data per voxel.
6. u_BlockTextures (Binding 6): sampler2DArray containing all block textures (16x16).
7. u_BlockData (Binding 7): isamplerBuffer storing 2 ivec4 per block ID:
   - [tex-Y, tex+Y, tex-Z, tex+Z]
   - [tex-X, tex+X, transparency, packed_refl_tint]
8. u_BiomeMap (Binding 8): sampler2D storing RG8 (Temp, Hum) values for the world.
9. u_GrassColormap (Binding 9): Minecraft grass colormap for tinting.
10. u_FoliageColormap (Binding 10): Minecraft foliage colormap for tinting.

--- BLOCK IMPLEMENTATION DETAILS ---
Model Resolution: Recursive parsing of 1.12.2 JSON models. Supports parent inheritance and face keys (top, bottom, side, end, up, down, etc.).
Face Mapping: Ordered indices: 0:-Y (bottom), 1:+Y (top), 2:-Z (north), 3:+Z (south), 4:-X (west), 5:+X (east).
UV Correction: X-axis faces (-X, +X) use swapped (Z, Y) mapping to fix 90-degree texture rotation.
Tinting: Automatically detected via "tintindex" in JSON. Shader samples u_BiomeMap and applies u_GrassColormap.

--- RECENT OPTIMIZATIONS & FIXES ---
- Crash-on-Missing: TextureManager and BlockDataManager now throw RuntimeException on any asset load/resolution failure.
- Reserved Word Fix: Renamed 'packed' to 'packedVal' in shader to avoid GLSL compiler errors.
- Recursive JSON: resolveModelRecursive follows parent chains to resolve textures and properties.
- Biome Map: Generated once at startup (2048x2048 RG8) using dual-seeded Perlin noise.
- Light Engine: Integrated LightPropagationEngine for packed RGB voxel lighting.
